# Supplementary Materials

This folder contains the study's supplementary materials. The files are intended to facilitate secondary analysis and improve the transparency of the reported SEM results.

## Contents

The folder includes:

- The analytic codebook
- Pooled and country-specific means and standard deviations
- Categorical frequency tables
- Pooled and country-specific covariance matrices
- Pooled and country-specific correlation matrices

These materials correspond to the observed indicators used in the corrected SEM analyses, including versions in which CSR1 is coded as 0 = No and 1 = Yes.

## Analytic Sample

The supplementary files correspond to the final analytic sample used in the SEM analyses:

- Total sample: N = 653
- Argentina: n = 144
- Colombia: n = 168
- Peru: n = 163
- Mexico: n = 178

## Purpose

These materials are provided to support transparency, reproducibility, and secondary analysis of the study findings.
