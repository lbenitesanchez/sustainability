# Sustainability Supplementary Materials

This repository contains the supplementary materials for the study and is intended to support transparency and facilitate secondary analysis.

## Repository structure

- `R-code/`: R scripts used to generate the supplementary outputs.
- `supplementary-data-and-codebook/`: supplementary files, including the analytic codebook, summary statistics, frequency tables, and pooled and country-specific covariance and correlation matrices.

## Notes

The files in `supplementary-data-and-codebook/` were produced using the R scripts stored in `R-code/`.

These materials correspond to the final analytic sample used in the corrected SEM analyses.
